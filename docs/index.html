О проекте

Туристический бот - это ваш персональный гид по интересным местам. Бот поможет вам:

  • Найти интересные места поблизости
  • Получить информацию о достопримечательностях
  • Прослушать аудиогиды
  • Совершить виртуальный тур

Чтобы начать использовать бота, просто нажмите на кнопку Telegram в правом нижнем углу!
